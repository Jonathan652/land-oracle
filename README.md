# Statum Legal Intelligence

**Statum Legal** is a professional, trilingual legal expert system designed specifically for the Ugandan jurisdiction. It providing authoritative statutory guidance, high-fidelity document drafting, and procedural analysis in **English, Luganda, and Runyankore**.

## ⚖️ Core Capabilities

- **Expert Statutory Analysis**: Real-time synthesis of the Ugandan Constitution, Statutes, and Subsidiary Legislation.
- **Automated Legal Drafting**: Generate professionally formatted Tenancy Agreements, Land Sale Agreements, Deeds of Assignment, and more in PDF or DOCX format.
- **ULII Judicial Integration**: Contextual retrieval of legal precedents and judicial rulings from the Uganda Legal Information Institute database.
- **Procedural Roadmaps**: Visual, step-by-step guides for complex legal processes like land registration, probate, and business incorporation.
- **Multilingual Support**: High-precision legal reasoning in native Ugandan languages (Luganda and Runyankore) alongside English.

## 🚀 Key Features

- **Professional Document Export**: One-click professional PDF generation for any legal advice or draft provided.
- **Interactive Voice Mode**: Real-time legal consultation via voice, optimized for accessibility.
- **Statutory Verification**: Multi-vector checks to ensure compliance with the latest 2023 revised laws of Uganda.
- **Advocate Connection**: Directly browse and connect with verified legal practitioners across various specialties.

## 🛠 Tech Stack

- **Reasoning Engine**: Multi-model routing (Gemini 3.1 Pro / Llama 3.3) for resilient, high-fidelity legal logic.
- **Frontend**: React 19 with Tailwind CSS 4.0 for a high-performance, responsive interface.
- **Backend**: Express.js server providing secure API orchestration and document processing.
- **Authentication & Database**: Firebase integration for secure session management and case history.

## 📦 Installation & Setup

1. **Clone the repository**
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Configure Environment Variables**: Create a `.env` file based on `.env.example`.
4. **Run in Development**:
   ```bash
   npm run dev
   ```
5. **Build for Production**:
   ```bash
   npm run build
   ```

## 📜 Disclaimer

Statum Legal is a legal intelligence system designed to assist with statutory research and document preparation. It provides information based on the Laws of Uganda (2023 Rev.). While it utilizes high-precision reasoning engines, the output should be reviewed by a qualified legal practitioner.
